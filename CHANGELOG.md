# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-16

### Added
- Initial release of Sleep Tracker application
- Sleep record management with date, hours, and quality tracking
- Daily habit tracking with completion status and notes
- Statistical dashboard with average sleep calculation
- Habit completion rate analysis
- JSON-based data persistence
- Comprehensive unit tests (30+ test cases)
- Checkstyle code quality analysis
- GitHub Actions CI/CD pipeline
- Professional documentation and README
- MIT License

### Features
- ✅ Add, view, and delete sleep records
- ✅ Create and track daily habits
- ✅ Calculate average sleep hours (7 and 30-day periods)
- ✅ View habit completion rates
- ✅ Persistent local data storage
- ✅ Intuitive JavaFX GUI interface
- ✅ Input validation and error handling

### Technical
- Built with Java 17 and JavaFX 21
- Maven build system
- JUnit 5 testing framework
- GSON for JSON serialization
- Cross-platform compatibility
